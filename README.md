# 🏎️ Ellie's Karting & Bumper Cars

A premium, feature-rich multiplayer racing and bumper car resource for **FiveM** (built for QBCore / Qbox). Featuring a high-end glassmorphic green dashboard, an interactive in-game track creator studio, real-time betting pools, and a robust multiplayer sync framework.

Developed by **Ellie** (Ellie West).

---

## ✨ Key Features

### 1. 🏁 Interactive Admin NPC Setup (`/setupgokart`)
* Walk to any location in-game and run the `/setup gokart` command to open the admin panel.
* Select from multiple ped models and idle scenarios.
* Click **"Preview Here"** to spawn a client-side preview ped to check heading and animations.
* Click **"Save & Apply"** to permanently persist coordinates to `npc_coords.json` on the server and spawn the coordinator NPC globally.

### 2. 💻 Modern Glassmorphic Green NUI Dashboard
* Premium translucent UI design tailored with neon-green styling.
* Displays a list of active lobbies, their tracks, host names, player counts, and statuses.
* Only admins can create lobbies (fully toggleable via `Config.AdminOnly`).
* Dynamic dropdown selection for different Kart models (Standard Kart, Veto Classic, Veto Modern, and Lambokart bumper cars).

### 3. 💸 Race Betting Pools
* Built-in cash betting system allowing lobby members to bet money on the race outcome.
* Live-updating bet roster displaying the contribution amount next to each racer's name in the lobby room.
* The winner automatically takes the accumulated cash pool on race completion, synced with server notifications.

### 4. 🛠️ In-Game Track Creator Studio
* Built-in visual editor accessible from the admin NUI tab.
* Displays a transparent green floating HUD bubble tracking starting grid positions and checkpoints.
* **Hotkeys**:
  * `[E]` - Place checkpoint cylinder (features custom sizes and distinct finish lines).
  * `[G]` - Place starting grid positions (captures player heading/heading vectors).
  * `[BACKSPACE]` - Undo last action.
  * `[ENTER]` - Save the custom track to the database.
  * `[ESC]` or `/canceltrack` - Exit without saving.
* Automatically syncs new custom tracks to all connected players instantly.

### 5. ⚡ Zero-Latency Multiplayer HUD & Sync
* AUTHORITATIVE telemetry sync keeps player positions up-to-date.
* The client locally increments lap counts and resets stopwatch timers when crossing the finish line to eliminate network lag.
* Highlights the local player as **"Self"** in the leaderboard card with a green border.

### 6. 🛡️ Lifecycle Cleanup & Recovery System
* Automatically cleans up spawned vehicles and returns players to the dashboard if they exit mid-race or type `/leaverace`.
* **DNF Timeout**: Starts a 20-second timeout when the first racer crosses the finish line, preventing stuck sessions due to AFK players.
* **Instant Scoreboard Dismissal**: Press ESC, Space, Enter, or click the results screen to instantly exit the scoreboard and clean up your character state.
* **"Force Leave" Recovery**: A sidebar button that allows desynced players to trigger an immediate server-side state reset and return safely to the browser.


## 🛠️ Quick Installation

1. Run the `database.sql` and `database_update.sql` schemas inside your database editor.
2. Drag and drop `gokart_racing` (along with dependencies `lambokart` and `mappingcarbumpers`) into your resources folder.
3. Add the following to your `server.cfg`:
```cfg
ensure gokart_racing
ensure lambokart
ensure mappingcarbumpers

# Resource permissions for automated ensuring/refreshes
add_ace resource.gokart_racing command allow
```
